Doom RPG Remake Project (DRRP)

This project brings the Doom RPG game from a mobile platform to the GZDoom engine.

The original atmosphere is beautifully intertwined with the latest features of the most modern Doom-ports.

Detailed levels, RPG-leveling system, many dialogs in two languages - all of this in the DRRP project. Проект DRRP переносит игру Doom RPG с мобильной платформы на движок GZDoom.

Оригинальная атмосфера красиво переплетается с новейшими возможностями современнейших Doom-портов.

Детализированные уровни, система RPG-прокачки, множество диалогов на двух языках - всё это в проекте DRRP.
